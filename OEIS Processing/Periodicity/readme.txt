infos about the file "periodic_infos.csv"

Do not open the file in excel, values are not correcly displayed

To load the file in python use:
	import numpy as np
	isPoly = np.loadtxt('periodic_infos.csv', delimiter=',')

First column: 	oeis_id of the sequence in question
Second column: 	Evaluation of the periodicity
		0 = no periodicity found
		1 = periodicity found
		3 = sequence of given numbers not long enough to establish at least 3 full periods
		4 = sequence is None, or has no elements
		5 = sequence is marked with "fini" or "dead" by oeis
Third column:	length of one period (shortest that was found)
Fourth column:	how many times the whole period was repeated in the given elements, if this is low it might indicate that the periodicity is not "real"
